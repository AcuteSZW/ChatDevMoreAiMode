# Gomoku Game

一个简单的五子棋游戏，支持两名玩家在15x15的棋盘上进行对战。

## 快速安装

确保您已经安装了Python 3.x。然后，您可以通过以下命令安装所需的依赖项：

```bash
pip install tkinter
```

## 🤔 这是什么？

Gomoku是一种经典的棋类游戏，玩家交替在棋盘上放置棋子，目标是率先在水平、垂直或对角线上连成五个相同的棋子。该应用程序提供了一个简单的图形用户界面，使玩家能够轻松进行游戏。

## 🎮 如何使用/播放

1. **运行游戏**  
   在终端中运行以下命令启动游戏：

   ```bash
   python main.py
   ```

2. **游戏界面**  
   游戏启动后，您将看到一个15x15的棋盘。玩家“X”和“O”将交替进行游戏。

3. **下棋**  
   点击棋盘上的任意一个按钮以放置您的棋子。游戏会自动检查是否有玩家获胜。

4. **游戏结束**  
   如果有玩家获胜，系统会弹出一个消息框显示获胜者，并重置游戏以便重新开始。
